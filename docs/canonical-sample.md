“Closest structure” and “cleanest edges” are auditable edge metrics, not
aesthetic rankings. Direction 4 was selected as the parent after visual review;
it also had this scout's highest structure similarity (`0.426114`) and edge
cleanliness (`0.989116`). Small characters, colors, and object identities are
still interpretations rather than exact reconstruction.

## Refine the selected parent

Direction 4 was varied at **Subtle** strength with **Structure locked**. All four
children preserve its broad arrangement while exploring paper texture, warm
accents, and background treatment.

remain evidence rather than an automatic art director.

The [scout manifest](assets/pocket-kingdom/manifest.json) and
[variation manifest](assets/pocket-kingdom/variation/manifest.json) contain
immutable model revisions, the resolved prompt and negative prompt, exact seeds,
parent lineage, technical scores, file hashes, and runtime versions. Studio's
current **Guided Sample** uses the separate privacy-reviewed HPO hero fixture;
Pocket Kingdom remains this documentation-only canonical lineage example.

!!! note "MPS reproducibility"

    Canny preserves edge layout, not semantic identity, exact counts, palette,
    or typography. MPS finite-output guards rejected unstable attempts during
    validation; no black or non-finite image is published. Hardware and runtime
    changes may preserve the recipe and seeds without producing pixel-identical
    output. Variation input is pre-encoded with the VAE in CPU fp32, and each
    MPS seed runs in an isolated pipeline to avoid cross-seed memory drift.
