# Fixed-seed Studio documentation fixture

This directory contains the previous fixed-seed four-direction study used for
the first English Studio capture. The current Hero fixture lives in
`../heritage-studio-hero-hpo-en/`. This study remains as compatibility and seed
comparison evidence; it is separate from the Pocket Kingdom Guided Sample
bundled in the Python wheel.

The selected first direction keeps the recorded 2023 seed
`6764547109648557242`. The next two seeds are the alternatives preserved in the
same notebook cell, and the fourth is an explicit adjacent value added to fill
the Studio's four-direction layout. Nothing in this fixture uses a random seed.

The English prompt is a new, human-written synthesis of the lesson brief and
the saved Korean runtime prompt. It is not presented as a recovered AWS
Translate string. The canonical `manifest.json` records the effective prompt,
negative prompt, seed list, Canny thresholds, model revisions, scheduler,
steps, CFG, ControlNet strength, runtime versions, and every artifact hash.

The source is the reviewed, metadata-free family-artwork derivative used by
the public v2 documentation. The legacy JPEG was not copied because it contains
camera and GPS EXIF. Do not infer or add a child's name to public documentation.
